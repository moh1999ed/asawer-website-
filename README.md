# موقع أساور للتطوير العقاري

موقع إلكتروني احترافي لشركة أساور للتطوير العقاري مبني باستخدام Next.js و TypeScript و Tailwind CSS.

## المميزات

- ✅ تصميم عصري ومتجاوب
- ✅ دعم كامل للغة العربية (RTL)
- ✅ صفحات متعددة: الرئيسية، من نحن، المشاريع، الخدمات، اتصل بنا
- ✅ أداء عالي وسريع
- ✅ SEO محسّن

## التقنيات المستخدمة

- **Next.js 15.5.6** - إطار عمل React
- **TypeScript** - للكتابة الآمنة
- **Tailwind CSS 4** - للتصميم
- **React 19** - مكتبة واجهة المستخدم

## التثبيت والتشغيل

### 1. تثبيت المتطلبات

```bash
npm install
```

### 2. تشغيل المشروع في وضع التطوير

```bash
npm run dev
```

افتح [http://localhost:3000](http://localhost:3000) في المتصفح.

### 3. بناء المشروع للإنتاج

```bash
npm run build
```

### 4. تشغيل المشروع بعد البناء

```bash
npm start
```
