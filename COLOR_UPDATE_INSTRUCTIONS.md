# تعليمات تحديث الألوان

تم تحديث جميع الألوان السوداء والرمادية في الموقع لتتوافق مع ألوان الشعار (التركوازي الفاتح والغامق).

## الخطوات المطلوبة:

1. **أوقف السيرفر** إذا كان يعمل (اضغط Ctrl+C)

2. **امسح التخزين المؤقت**:
```bash
cd asawer-website
rm -rf .next
rm -rf node_modules/.cache
```

3. **أعد بناء المشروع**:
```bash
npm run build
```

4. **أعد تشغيل السيرفر**:
```bash
npm run dev
```

5. **في المتصفح**:
   - اضغط `Ctrl+Shift+R` (Windows/Linux) أو `Cmd+Shift+R` (Mac) لتحديث قوي
   - أو افتح Developer Tools (F12) → Network → ✅ Disable cache → ثم أعد تحميل الصفحة

## الألوان المحدثة:

- **الغامق**: `#003135` (Deep Royal Turquoise)
- **الفاتح**: `#20B2AA` (Vibrant Mint Teal)

جميع النصوص والحدود والخلفيات الرمادية تم تحديثها لتستخدم هذه الألوان.
