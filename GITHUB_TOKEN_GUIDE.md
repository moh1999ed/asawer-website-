# GitHub Personal Access Token Guide

## Steps to Create a Token:

### Step 1: Go to GitHub Settings
Open this link directly:
```
https://github.com/settings/tokens
```

### Step 2: Create New Token
1. Click on the blue **"Generate new token"** button
2. Or click **"Generate new token (classic)"** if options appear

### Step 3: Token Settings
- **Note**: Enter any name like "asawer-website"
- **Expiration**: Choose duration (90 days or No expiration)
- **Select scopes**: Choose ✅ **repo** (Full control of private repositories)

### Step 4: Generate Token
- Click **"Generate token"** at the bottom
- ⚠️ **Very Important**: Copy the Token immediately as it won't be shown again!

### Step 5: Using the Token
When pushing code, use:
- **Username**: moh1999ed
- **Password**: Paste the Token (not your regular password)

3. Or go to: Settings → Developer settings → Personal access tokens → Tokens (classic)
