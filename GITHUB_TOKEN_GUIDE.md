# دليل إنشاء Personal Access Token على GitHub

## الطريقة الصحيحة:

### الخطوة 1: اذهب إلى إعدادات GitHub
افتح هذا الرابط مباشرة:
```
https://github.com/settings/tokens
```

### الخطوة 2: إنشاء Token جديد
1. اضغط على الزر الأزرق الكبير **"Generate new token"**
2. أو اضغط على **"Generate new token (classic)"** إذا ظهرت لك خيارات

### الخطوة 3: إعدادات Token
- **Note**: اكتب أي اسم مثل "asawer-website"
- **Expiration**: اختر المدة (90 يوم أو No expiration)
- **Select scopes**: اختر ✅ **repo** (كامل - Full control of private repositories)

### الخطوة 4: إنشاء Token
- اضغط **"Generate token"** في الأسفل
- ⚠️ **مهم جداً**: انسخ الـ Token فوراً لأنه لن يظهر مرة أخرى!

### الخطوة 5: استخدام Token
عند رفع الكود، استخدم:
- **Username**: moh1999ed
- **Password**: الصق الـ Token (ليس كلمة المرور العادية)

---

## رابط مباشر:
https://github.com/settings/tokens/new

## إذا لم تجد الزر:
1. تأكد من تسجيل الدخول إلى GitHub
2. جرب الرابط المباشر أعلاه
3. أو اذهب إلى: Settings → Developer settings → Personal access tokens → Tokens (classic)
